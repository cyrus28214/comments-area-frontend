# comments-area-frontend

X-Lab训练项目，评论区（前端）

![UI](./UI.pdf)

后端项目地址：[comments-area](https://github.com/cyrus28214/comments-area)

# 启动

1. 修改`/src/api/comment.js`中的`baseURL`到后端地址。

2. 启动服务：
```bash
cd comments-area-frontend
pnpm dev
# 或者先build
# pnpm build
```
